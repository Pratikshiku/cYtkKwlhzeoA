- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 论坛
5. 歌曲
6. 留言板
7. 公告信息
8. 个人中心
9. 歌曲收藏
10. 修改密码
11. 个人信息
12. 论坛管理
13. 歌曲管理
14. 歌曲留言管理
15. 歌曲收藏管理
16. 留言板管理
17. 公告信息管理


**管理员功能：**
1. 登录
2.个人中心（修改密码、修改信息）
3. 管理员管理
4. 歌曲类型管理
5. 公告类型管理
6. 论坛管理
